## 📝 使用说明

### 在新环境中使用

如果你在其他电脑上克隆这个仓库：

```bash

git clone https://github.com/你的用户名/feishu-ai-bot.git
cd feishu-ai-bot

# 2. 复制配置模板
copy .env.example .env  # Windows
```
