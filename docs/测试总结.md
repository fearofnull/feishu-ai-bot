## 技术细节

### 消息去重机制
使用Python的`collections.deque`实现：
```python
processed_message_ids = deque(maxlen=1000)
```
- 自动维护最近1000条消息ID
- 超出容量时自动移除最早的消息ID
- 防止重复处理同一条消息

### Claude Code CLI调用
- 使用subprocess.run执行命令
- 通过stdin传递用户输入
- 设置600秒超时
- 捕获stdout和stderr输出
- 在目标项目目录中执行

### Git操作
